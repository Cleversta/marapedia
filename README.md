# Marapedia

A multilingual cultural encyclopedia for the Mara people.

## Features
- 4 languages: Mara, English, Myanmar, Mizo
- Flutter mobile app (published on Google Play)
- Next.js web app
- Supabase backend + Cloudflare R2 storage
- Rich content editor
- Offline reading support

## Tech Stack
- Flutter + BLoC
- Next.js + TypeScript
- Supabase
- Cloudflare R2
